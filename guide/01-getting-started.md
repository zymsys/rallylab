# Chapter 1: Getting Started

## 1.1 Login Screen

The login screen is the entry point to RallyLab. You can sign in with your email using a magic link, or try the demo mode to explore all features with sample data.

![Login Screen](images/pre-race-login.png)

## 1.2 Demo Mode

Click "Try Demo" to load sample rally data and explore all features without signing in. Demo mode creates a sample rally with three sections (Beaver Buggies, Kub Kars, Scout Trucks), groups, and participant rosters.

![Demo Mode — Rally List](images/pre-race-rally-list.png)
